# Universal Scraping Architect
## Purpose
This skill is a reusable, general-purpose scraping and data extraction framework.
Use it whenever the user asks for scraping, crawling, extraction, parsing, or data collection.

## Mandatory Approach Selection Rule
1. **Firecrawl** — public web data, dynamic content, search-first discovery
2. **Traditional/local scraping** — local files, private data, static HTML
3. **Hybrid approach** — Firecrawl for web extraction, local tools for processing

## Master Rule
The scraping framework must always remain customizable, approach-aware, token-aware, Firecrawl-quota-aware, validated, and error-safe.
